HTMl常用标签

November 15, 2019

大家好，今天我们学习了HTML常用的几个标签

1. a 标签的用法

首先我们了解下 a 标签：链接通过 a 标签表示，用户点击后，浏览器会跳转到指定的网址。 今天主要学习了它的三个属性： 1. href属性:给出链接指向的网址。它的值应该是一个 URL 或者锚点。 2. target属性:指定如何展示打开的链接。它可以是在指定的窗口打开，也可以在iframe标签里面打开。 3. download属性:表明当前链接用于下载，而不是跳转到另一个 URL。

其次是 a 的 href 的取值：

  • 网址 : 建议使用//google.com
  • 路径 : /a/b/c以及a/b/c
  • 伪协议 : javascript:代码; 或 mailto:邮箱 或 tel：手机号
  • id : href=#xxx

2. img 标签的用法

今天我们学习的 img 标签用法如下：img 标签用于插入图片。它是单独使用的，没有闭合标签。

  • 作用 : 发出get请求，展示一张图片。
  • 属性 : alt(设定图片的文字说明)/height(指定图片显示时的高度)/ width(指定图片显示时的宽度和高度)/src(图片的来源)
  • 事件 : onload/onerror
  • 响应式 : max-width:100%

3. table 标签的用法

我们再来回顾一下 table 标签的用法，相关的标签有：

  • table : 块级容器标签，所有表格内容都要放在这个标签里面
  • thead : 块级容器标签，是table的一级子元素，表示表头。
  • tbody : 块级容器标签，是table的一级子元素，表示表体。
  • tfoot : 块级容器标签，是table的一级子元素，表示表尾。
  • tr : 表示表格的一行（table row）。
  • td : 定义表格的数据单元格。
  • th : 定义表格的标题单元格。

相关的样式有:

  • table-layout
  • border-collapse
  • border-spacing
